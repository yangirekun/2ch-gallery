# Загрузчик изображений и галерея для одного оранжевого сайта с колобками.

## Разработка:

`yarn` <br>
`yarn start:dev` <br>

При попытке сделать коммит, будет произведен:

## Билд

`yarn build`

## И прогон тестов

`yarn test`

## Для деплоя изменений:

`git push heroku master`

## Актуальная версия:

`https://tranquil-beach-77502.herokuapp.com/`
